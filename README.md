BATCH RENDER 2.0 - INSTALLATION GUIDE
========================================

Follow these steps to correctly install the script and extension for VEGAS Pro:

1. INSTALLING THE SCRIPT (.cs)
------------------------------
Copy the file "Batch Render 2.0.cs" to ONE of the following locations:

Option A (Specific to Vegas Pro 2026):
C:\Program Files\BorisFX\Vegas Pro 2026\Script Menu

Option B (Global Script Folder):
C:\ProgramData\VEGAS Pro\Script Menu


2. INSTALLING THE EXTENSION (.dll)
----------------------------------
Copy the file "BatchRenderExtension.dll" to the following location:

Path:
C:\ProgramData\VEGAS Pro\Application Extensions


3. HOW TO USE
-------------
1. Restart VEGAS Pro if it's currently running.
2. To run the script: Go to "Tools" -> "Scripting" and select "Batch Render 2.0".
3. The extension (if applicable) will be available under "View" -> "Extensions".

----------------------------------------
Note: If you cannot find the "ProgramData" folder, make sure "Hidden items" is checked in your Windows File Explorer view settings.
